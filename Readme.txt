NodeJS/socketio based Signaling Server for the iConfRTC SDK
To run, open a command line prompt, go to the folder of the server.js and type node.exe server.js then press enter.
The signaling server will start listeningon the default port of 8080.

To change the listening port, simply pass the port number as a parameter.

Example to run the signaling port on tcp port 8081 :

> node.exe server.js 8081
